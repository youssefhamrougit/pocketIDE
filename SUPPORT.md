# Getting Support

Welcome to PocketIDE! Here's how to get help depending on what you need.

## 📖 Documentation

Check the [README](README.md) and the [Project Plan](Plan.md) first — they cover the vision, features, and architecture.

## 💬 Community Discussions

For questions, ideas, and general discussion:

- **GitHub Discussions** — [Start a discussion](https://github.com/youssefhamrougit/pocketIDE/discussions)
- Ask questions, share plugins, show off your setup, or brainstorm features

## 🐛 Bug Reports

Found a bug? [Open a bug report](https://github.com/youssefhamrougit/pocketIDE/issues/new?template=bug_report.md).

Before reporting:
- Search [existing issues](https://github.com/youssefhamrougit/pocketIDE/issues) to see if it's already known
- Include your device model, OS version, and app version

## ✨ Feature Requests

Have an idea? [Submit a feature request](https://github.com/youssefhamrougit/pocketIDE/issues/new?template=feature_request.md).

## 🔌 Plugin Help

For issues with a specific plugin, contact the plugin author first. If you believe the plugin system itself has a bug, open a regular issue.

## 🛡️ Security Issues

**Do not** open a public issue. Report privately via [our security policy](SECURITY.md).

## 💼 Commercial / Partnership Inquiries

For business inquiries, partnerships, or sponsorship opportunities, reach out via youssefmeddebhamrouni@gmail.com.
